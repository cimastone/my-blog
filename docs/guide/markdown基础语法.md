# Markdown基础语法

## 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式
```markdown
**粗体**
*斜体*
~~删除线~~
```

## 链接和图片
```markdown
[链接文本](URL)
![图片描述](图片URL)
```

## 列表
```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

## 代码
```markdown
`行内代码`

```代码块```
```

## 表格
```markdown
| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
```
