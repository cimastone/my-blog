本文基于 NVIDIA（NVDA）财年 2025（截至 2025-01-26）与财年 2026（截至 2026-01-25）的 10-K 数据，整理库存拨备（inventory provisions）与库存周转天数（DIO）的变化，并给出可证伪的风险跟踪框架，目的是把“库存增长”进一步拆解为**增长性备货**还是**压力型库存**。

## 1. 已知事实（Facts）

- 库存拨备（inventory provisions）
  - 2026：**4.0B**
  - 2025：**1.6B**
  - 绝对金额约 **2.5x** 增长。

- 成本（COGS / cost of revenue）
  - 2026：**62.475B**
  - 2025：**32.639B**

- 期末库存（Ending inventory）
  - 2026：**21.403B**
  - 2025：**10.080B**

- DIO（使用 cost of revenue / 平均库存的“平均余额口径”）
  - 2025：**85.99 天**
  - 2026：**91.96 天**
  - 库存消耗天数变长（周转变慢）。

> 注：DIO 计算口径：  
> DIO = 365 / (COGS / Avg Inventory)  
> 其中 Avg Inventory = (期末库存 + 上一年期末库存) / 2。

## 2. 两个“拨备强度”指标（把绝对值转成可比口径）

为了避免只看拨备绝对金额（会被规模扩张放大），引入两个可迁移指标：

### 2.1 Inventory provision / COGS

- 2026：4.0 / 62.475 ≈ **6.4%**
- 2025：1.6 / 32.639 ≈ **4.9%**

结论：拨备强度（相对成本）**小幅上升**（约 +1.5pct）。

### 2.2 Inventory provision / Ending inventory

- 2026：4.0 / 21.403 ≈ **18.69%**
- 2025：1.6 / 10.080 ≈ **15.87%**

结论：拨备强度（相对库存）**小幅上升**（约 +2.8pct）。

## 3. 初步判断：更偏“增长性备货”，但库存质量边际走弱

- 期末库存增长倍数约 **2.12x**（21.403/10.080）
- COGS 增长倍数约 **1.91x**（62.475/32.639）
- 拨备强度仅温和上升（并未与绝对金额一样“失控式上升”）

因此当前更像：在业务规模快速扩张下，库存与成本一起抬升；但 **DIO 上升 + 拨备强度上升** 表明库存质量出现边际走弱，需要持续验证是否会转为压力型库存。

## 4. 风险与可能原因（需要与 10-K 原文进一步对照）

结合 NVDA 的风险披露语境，库存增加及拨备上升可能与以下因素相关（后续需要用 10-K 原文逐条引用验证）：

- 出口限制/许可证要求等监管变化，导致需求节奏与交付结构变化
- 对未来需求的评估偏差（forecast error）
- 产品迭代与更新频繁导致部分存货过时/降价
- 上游供给节奏与需求节奏不匹配（lead time mismatch）

## 5. 可证伪框架（未来跟踪指标）

为了把“增长备货”与“压力库存”区分开，建立一组证伪条件。若未来连续出现以下信号，则本篇“偏增长性备货”的判断应被下修：

1) **库存增速持续高于 COGS / Revenue 增速**（DIO 继续上升）
2) **拨备强度继续上行**（Provision/COGS、Provision/Inventory 连续上升）
3) **毛利率出现非预期走弱**（库存写down计入 COGS，会压毛利）
4) **经营现金流持续被库存占用拖累**（CFO 中 Inventories 项连续大额为负）
